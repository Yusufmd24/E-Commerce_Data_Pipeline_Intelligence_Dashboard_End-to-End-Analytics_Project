# 💼 E-Commerce Data Pipeline & Intelligence Dashboard

🚀 **End-to-end analytics project transforming raw e-commerce data into actionable business insights using SQL, Python, and Power BI.**

---

# 📌 Problem Statement

E-commerce businesses generate large volumes of transactional data, but without proper analysis, this data fails to translate into meaningful decisions.

This project aims to answer:

* What drives revenue growth?
* Which products and categories dominate sales?
* Are customers truly loyal or just repeatedly purchasing due to limited options?
* Where are the hidden risks in the business?

---

# 🧱 End-to-End Workflow

Data Collection → Data Cleaning → Data Transformation → Data Modeling → Visualization → Insights
```

---

## 🔹 1. Data Collection (Python)

Raw data is collected and structured using Python.

* Extracted datasets via scripts
* Ensured consistent schema for analysis

---

## 🔹 2. Data Cleaning (Excel)

Initial cleaning and validation performed using Excel.

* Removed duplicates
* Standardized formats
* Quick exploratory checks

---

## 🔹 3. Data Transformation (SQL)

Data transformed into analysis-ready format using Microsoft SQL Server

* Joined multiple tables (Orders, Customers, Products)
* Aggregated key metrics
* Created structured datasets for reporting

---

## 🔹 4. Data Modeling (Power BI)

Built a star schema model in Microsoft Power BI

* Fact table: Orders
* Dimension tables: Customers, Products, Date
* Enabled efficient filtering and relationships

---

## 🔹 5. Data Analysis (DAX)

Implemented key business metrics:

* Total Revenue
* Average Order Value (AOV)
* Revenue Growth % (MoM)
* Repeat Customer Rate
* Revenue per Customer

---

## 🔹 6. Data Visualization (Power BI)

Designed an interactive dashboard with:

* KPI cards for quick insights
* Revenue trend analysis
* Customer behavior breakdown
* Product performance evaluation

---

# 📊 Dashboard Overview

## 1️⃣ Sales Performance

* Tracks revenue trends and growth patterns
* Identifies top-performing categories and products

## 2️⃣ Customer Insights

* Analyzes retention and repeat purchase behavior
* Highlights geographic distribution

## 3️⃣ Product & Pricing Insights

* Evaluates product-level contribution
* Analyzes rating vs revenue relationship

---

# 🧠 Key Insights

* Revenue shows consistent growth but is heavily dependent on the Furniture category
* Top 5 products contribute over 90% of total revenue → high concentration risk
* Extremely high repeat customer rate suggests strong retention but potential low acquisition
* Premium products dominate revenue, indicating strong high-value purchasing behavior

---

# 🚨 Business Risks Identified

* Over-reliance on a few products
* Category concentration risk
* Limited diversification
* Potential imbalance between retention and acquisition

---

# 🚀 Strategic Recommendations

* Diversify product portfolio
* Expand into underperforming categories
* Strengthen customer acquisition strategies
* Improve data collection for deeper funnel analysis
